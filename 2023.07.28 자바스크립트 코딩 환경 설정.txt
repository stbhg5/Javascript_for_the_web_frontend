[2023.07.28 자바스크립트 코딩 환경 설정]

1. 자바스크립트 동작 방식
자바스크립트는 자바스크립트 엔진이 들어있는 곳에서 동작하고, 자바스크립트 엔진은 웹브라우저에 포함되어있고, 이런 웹브라우저를 자바스크립트 실행환경, 런 타임 이라고 부른다.

2. 브라우저에서 Hello World 출력
Chrome 브라우저의 개발자 도구에서 콘솔을 통해 "hello world" 출력

(크롬 콘솔 출력)

console.log 명령어를 웹브라우저에 있는 자바스크립트 엔진이 해석해서 잘 실행한 것을 확인할 수 있으나, 코드가 즉시 실행이 되어 불편하다.
웹브라우저의 개발자 도구 보다 조금 더 편리하게 자바스크립트 코드를 실행시킬 수 있는 에디터에서 자바스크립트를 실행.

3. 코드 샌드박스 에디터에서 Hello World 출력
코드 샌드박스(CodeSandBox)는 환경설정과 별도의 설치 과정 없이 자바스크립트를 실행시킬 수 있도록 도와주는 온라인 에디터이다. (https://codesandbox.io/)
index.js 파일 만든 후 "hello world" 출력
